Welcome

C++ is a multi-paridigm, staticly typed, compiled programming language created by Bjarne Stroustrup. It was created using C and was originaly called C with classes.

C++ is used in a varity of software such as:

  • Video Games
  • Embedded Systems
  • Finacial Software
  • GUI
  • Windows Applications

History

Bjarne Stroustrup created C++ at Bell Labs and got the idea while working on his thesis with the Simula 67. It added oop concecpts to C and was considered a superset of C. It's name was changed to c++ in 1985.

Variables

Variables are parts of memory used to store a value. You can think of them as storage boxes. Variables are given names known as identifiers. You can name what whatever you want, with a few exceptions.

                
                    x = 5;
                    y = 7;

                    sum = x + y;

Conditionals

A conditional is a statement that if evaluates to true will execute it's block of code. An else statement will execute if none of the above statements are true. 

                
                    if(x > 4)
                    {
                        return true;
                    }
                    else
                    {
                        return false;
                    }

You can have multiple Conditionals using else if.

                    
                        if(x > 4)
                        {
                            x++;
                        }
                        else if(x < 4)
                        {
                            x--;
                        }
                        else
                        {
                            return true;
                        }

Loops

There are three types of loops. A For, While, and Do-While loop.

The While loop continues looping until the condition evaluates to true

                
                    while(x > 4)
                    {
                        x++;
                    }

The above code adds one to x until it is greater than 3.

The Do While loops operates the same except the condition is evaluted after a loop. The Do while loop is guaranteed to run once. 

                
                    do{
                        x++;
                    }while(x > 5);

The For loop iterates a certain number of times.

  • A variable is initialized at the start of the loop.
  • If the condition is true the loop continues else it stops.
  • The variable is the increased.
  • The loop repeats step 2.
                
                    int end = 9;

                    for(int start = 0; start != end; start++)
                    {
                        std::cout << start;
                    }
